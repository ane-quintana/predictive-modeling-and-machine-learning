# predictive-modeling-and-machine-learning

Collection of machine learning and data science projects focused on predictive modeling, neural networks, and business analytics using Python and Jupyter Notebook. Completed as part of MBA coursework at IESE Business School.

## Projects included

• AirBnB Data Science Project: exploratory data analysis, feature engineering, and predictive modeling to identify key drivers of pricing and demand.

• Neural Networks Project: implementation and evaluation of neural network models for predictive tasks using structured datasets.

## Tools and technologies

Python, Jupyter Notebook, Pandas, NumPy, Scikit-learn, TensorFlow/Keras, data visualization, predictive modeling
